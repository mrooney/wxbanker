Pre-requirements:
 * Ubuntu/Debian: sudo apt-get install python-wxgtk3.0 python-cairo # or python-wxgtk2.8 for older distros
 * OSX: brew install wxpython py2cairo

Installation:
 * pip install -r pip_requirements.txt

Usage:
 * python main.py
